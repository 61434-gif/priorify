# Priorify


Priorify est une application web de gestion de tâches intelligente.

## Objectif
Aider les utilisateurs à organiser leurs tâches selon la priorité, l'urgence et les dates limites.

## Stack technique
- Backend : Java, Spring Boot
- Frontend : React
- Base de données : PostgreSQL
- Authentification : Spring Security + JWT

## Méthodologie
Le projet est suivi avec une approche Agile sur GitHub :
- Issues
- GitHub Project
- Milestones
- Sprints

## Structure
- `backend/` : API Spring Boot
- `frontend/` : interface React
- `docs/` : documentation projet
